Elements Of Html

There are three types of Html Elements they are:

1.BLOCK LEVEL ELEMENTS

2.INLINE BLOCK ELEMENTS

3.INLINE ELEMENTS

1.BLOCK LEVEL ELEMENTS:

These Elements occupies whole width and creates the new line.

EX: Div,h1-h6,p,semantic elements etc.

2.INLINE BLOCK ELEMENTS:

The elements which occupies only required space and does not creates a new line and we can apply width,height,margin and padding properties is known as Inline block elements.

EX: image,iframe,input,button etc

3.INLINE ELEMENTS:

The elements which occupies only required space and does not create a new line. In CSS we cannot apply width, height,padding top and bottom and margin top and bottom.

EX: span,formatting tags,b,i etc.
